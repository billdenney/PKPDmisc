calculate basic NCA parameters t1/2 Cl, Cmax etc

Usage

nca(.time, dv, dose, last_times = c(3, 4, 5), digits = 2)

NCA(...)

Arguments

.time
vector of times
dv
vector of observations (concentrations)
dose
vector or single value of Dose given
last_times
vector of numbers of time points to evaluate for AUCinf extrapolation default 3-5
digits
number of digits to round to. Can use NULL for no rounding
...
arguments to pass to nca

Description

calculate basic NCA parameters t1/2 Cl, Cmax etc

Details

Works well in tandem with the dplyr package and `do` verb
