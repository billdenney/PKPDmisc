Convert '.' values into missing values.

Usage

replace_dots(x)

Arguments

x
A character vector

Value

A character vector with '.' replaced by NA values.

Description

Convert '.' values into missing values.

Details

good for nonmem preprocessing as missing values are represented with '.' in NONMEM

Examples

x <- c(".", "1", "1")
replace_dots(x)


[1] NA  "1" "1"
