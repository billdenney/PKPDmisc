read xpose tables to xpose database and save and compress as RDS

Usage

read_xpdb(runno, table.names = c("sdtab", "mutab", "patab", "catab", "cotab",
  "mytab", "extra", "xptab", "cwtab"), RDS_name = paste0("xpdb", runno),
  rm_tabs = FALSE, ...)

Arguments

runno
run number
table.names
table names for xpose.data to search for
RDS_name
name of RDS save file
rm_tabs
logical value to delete output tables after saving to xpdb
...
arguments to pass to xpose.data

Description

read xpose tables to xpose database and save and compress as RDS
