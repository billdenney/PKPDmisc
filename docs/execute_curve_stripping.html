curve strip a dataframe with multiple IDs

Usage

execute_curve_stripping(df, TIME_name = "TIME", DV_name = "CONC",
  DOSE_name = "DOSE", number_terminal_points, ...)

Arguments

df
dataframe to curve strip
TIME_name
column name for time column
DV_name
column name for dependent variable column eg. conc
DOSE_name
column name for dose column
number_terminal_points
number of data points in terminal phase
...
additional args to be passed to strip_curves

Description

curve strip a dataframe with multiple IDs

Details

calculates the geometric mean by dose group then curve strips each mean expects 3 columns will be in dataset, the independent variable (eg time), dependent variable (eg concentration), and a dose column with the dose given for each ID in all rows
