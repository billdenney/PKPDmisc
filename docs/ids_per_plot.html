split IDs into groups to use for subsequent plotting

Usage

ids_per_plot(id, id_per_plot = 9)

Arguments

id
vector of ids (eg id column)
id_per_plot
number of ids per plot. Default to 9

Description

split IDs into groups to use for subsequent plotting

Details

works very well with hadley wickham's purrr package to create a column to split on then subsequently plot
